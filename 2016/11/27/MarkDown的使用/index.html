Markdown 学习笔记

Markdown 介绍

Markdown是一种可以使用普通文本编辑器编写的标记性的语言，换句话说，就是一种轻量级的标记语言，
通过简单的标记语言，可以让普通文本的内容具有一定得格式。

Markdown 用途

  • Markdown的语法简洁明了、学习起来很容易
  • 其功能比纯文本更强
  • 可读性， Markdown 可以选择性的转换为 HTML 文档格式

Markdown 优点

  • 纯文本，所以兼容性极强，可以用所有文本编辑器打开
  • 专注于文字写作而不是排版
  • 格式转换方便，可以轻松转成HTML、PPT、电子书等
  • Markdown标记语法简单好记，可读性强

Markdown 常用语法

  1. 标题：标题是每篇文章都需要也是最常用的格式，在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加 # 号即可。
    标题就用#来表示。建议在井号后加一个空格，这是最标准的 Markdown 语法

  2. 段落：
    一个回车之后再两个空格是换行，但是还是一个段落 ，两个回车是另一个新的段落，前后是不加东西的

  3. 列表：在 Markdown 下，列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加 1. 2. 3.
    符号要和文字之间加上一个字符的空格。

  • 一级列表用’-‘来表示
    • 二级嵌套子列表使用’+’,是两个缩进
      • 三级列表使用’*’,在二级列表的基础上，也要两个缩进
  1. 引用：
    如果需要引用一小段别处的句子，那就要使用引用的语法格式 ，用英文下的>。
    例子如下：

    床前明月光，疑是地上霜
